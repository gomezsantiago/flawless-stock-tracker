## Getting Started

- [Install npm with Node](https://nodejs.org/en/download/)
- Update local repo

```sh
npm update
```

- Run development start script

```sh
npm run start
```

- The application is now accessible on localhost:8080

## Running Tests

- Run tests with this command :

```sh
npm run tests
```
